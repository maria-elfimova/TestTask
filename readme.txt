Способ работы с программой:
1) Собрать проект Visual Studio и запустить программу.
2) С помощью соответствующих кнопок выбрать картинку и файл для сохранения картинки в текстовом формате.
3) Запустить конвертирование, нажав на соответствующую кнопку.
4) Программа сообщит, когда конвертирование будет закончено.
